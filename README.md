# MindPass Bot

Телеграм‑бот для генерации запоминающихся, но безопасных паролей. Бот строит пароль из нескольких коротких русских слов, случайных цифр и специальных символов и позволяет быстро перегенерировать результат, прежде чем сохранить его.

## Возможности
- Команда `/start` открывает главное меню с кнопками.
- Команда `/help` выводит подробную справку.
- Кнопка **«Сгенерировать пароль»** запускает диалог: бот спрашивает тег, генерирует пароль, позволяет перегенерировать и сохранить его.
- Пароль отправляется скрытым сообщением (MarkdownV2) — для отображения нужно тапнуть по тексту.

## Требования
- Python 3.9+.
- Зависимости из `requirements.txt` (включая `aiogram` 2.25.2).
- Файл `russian.txt` в корне проекта с короткими русскими словами (по умолчанию используются слова длиной до 5 символов). Файл не входит в репозиторий и должен быть подготовлен самостоятельно.
- Токен Телеграм‑бота.

## Установка и запуск
1. Клонируйте репозиторий и перейдите в каталог проекта.
2. Создайте виртуальное окружение (необязательно, но рекомендуется) и установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Подготовьте файл `russian.txt`:
   ```bash
   echo -e "кот\nдом\nмост\nлуг" > russian.txt
   ```
4. Установите токен бота. Самый простой способ — задать переменную окружения и читать её из кода. Например, добавьте в начало `mindpass.py` чтение токена из `BOT_TOKEN = os.getenv("BOT_TOKEN")` и экспортируйте значение перед запуском:
   ```bash
   export BOT_TOKEN="<ваш_токен_бота>"
   ```
   > В репозитории присутствует тестовый токен, замените его на собственный перед использованием.
5. Запустите бота:
   ```bash
   python mindpass.py
   ```

## Как работает генератор
- Список слов загружается из `russian.txt`, слова транслитерируются в латиницу и капитализируются.
- Пароль собирается из двух слов, двух цифр и одного спецсимвола (набор ограничен символами `~!@#$&*(){}[]+"':<>/%`).
- Порядок компонентов перемешивается перед отправкой пользователю.
